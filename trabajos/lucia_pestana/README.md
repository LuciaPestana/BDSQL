## 📋 Descripción
Análisis completo de la base de datos **BDventas**, 
un sistema de gestión de clientes y comerciales 
con seguimiento de comisiones.

## 💾 Base de datos elegida
**BDventas** — Base de datos de ventas con MySQL

## 📊 Tablas
| Tabla | Descripción |
|-------|-------------|
| `clientes` | 👤 Datos personales de los clientes |
| `comercial` | 💼 Datos de los comerciales y sus comisiones |
| `auditoria` | 🔎 Registro de cambios de comisiones |
